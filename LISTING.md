## Summary
Provides easy keyboard shortcuts to open new tabs and windows on different containers: Ctrl+Shift+[1-9] / or Alt+Shift+[1-9] on Mac.

## Description
Provides easy keyboard shortcuts to manage containers in tabs and windows:

Search Containers: `Ctrl + Shift + X` opens a popup where you can type the name of a container and open a new tab in it. Use arrow keys to navigate and `Enter` to open a tab, or `Ctrl + Enter` to open in a new window.

Open a new tab with a given container: `Ctrl + Shift + [1-9]` (or `Alt + Shift + [1-9]` on Mac).

Open a new tab on the current container: `Ctrl (Cmd on Mac) + Alt + T`.

Reopen current tab with specified container: `Ctrl (Cmd on Mac) + Alt + [1-9]`.

Reopen current tab in default container: `Ctrl (Cmd on Mac) + Alt + 0`.

New Window on Current Container: `Alt (Ctrl on Mac) + Shift + T`.

New Window on Specified Container: `Alt (Ctrl on Mac) + Shift + [1-9]`.
